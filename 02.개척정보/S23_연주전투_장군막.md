# 연주전투 장군막 공략

## 0) 결론

1) 장군막이 처음 열리면 풍을 고른다.
2) 군책-정 에서는 징병소모자원 감소와 군영 건설비용 감소를 고른다.
3) 풍계 장군령에서 석재 구입을 우선으로 한다.
4) [옵션] 3사령단 이상 건설이 필요한 경우 잠시 장군막을 림으로 바꾸고, 사령단 건설 후 다시 풍으로 변경한다.
5) 쟁 중에는 화 장군막을 유지한다.
6) [옵션] 보조 역할을 수행하는 계정은 쟁중에 산 장군막을 선택한다.

## 1) 장군막의 구성

1) 장군막은 장군령 / 군책으로 이루어져 있다.
2) 군책은 자원 / 군사스킬로 이루어져 있다.
3) 장군막은 "풍/림/화/산" 4가지 분류로 나뉘어지며, 각각의 등장시점은 다음과 같다.
    * 풍, 림 : 사건 "복양 포위" 완료 후 오픈(기서여림, 서버 오픈 3~4일차)
        * 동시 오픈 기능 : 병서, 출사, 1번째 치군, 전우, 군사기능
    * 화 : 사건 "마릉 매복" 완료 후 오픈(맹렬한 침략, 복양 포위 완료 후 5일 뒤, 서버 오픈 8~9일차)
        * 동시 오픈 기능 : 2번째 치군
    * 산 : 사건 "임시 휴전" 완료 후 오픈(부동여산, 마릉 매복 완료 후 5일 뒤, 서버 오픈 13~14일차)
        * 동시 오픈 기능 : 호상삼군 보급량 증가

## 2) 장군령

1) 장군령은 자원 유형 / 군사 스킬 유형 으로 이루어져 있다.

2) 장군령은 "풍/림/화/산"에 대응하여 각각 "석재/동전/식량/예비병" 자원을 바꾸거나, 각 유형에 대응하는 군사 스킬을 사용할 수 있다.

3) 장군령은 "병부"를 소모하여 사용할 수 있으며, 시스템 오픈 초기 기본 30병부를 제공하며, 30분에 1병부가 리필된다. -> 24시간동안 48병부가 리필되며, 상한은 50병부이다.

4) 장군막에서 속성(풍림화산)을 선택하면, 최초의 장군령은 4개가 등장하며, 새로운 장군령을 사용하려면 4개의 장군령을 모두 사용하거나, 장군막의 속성을 변경해야 한다.
    * 장군막의 속성은 반드시 다른 속성으로만 교체가 가능하다.(ex. 풍 -> 림)

## 3) 군책

1) 군책은 군책-정 / 군책-기로 이루어져있다.

2) 군책-기는 "풍림화산" 어느것을 골라도 동일하게 3종류가 존재한다.

    * 포로 획득 / 특수 스킬 보물 / 보좌 명령 사용

3) 군책-정은 장군막의 "풍림화산" 선택에 따라 서로 다른 효과를 사용할 수 있으며, 해당 분야의 2가지 군책을 선택할 수 있다.

4) 군책의 변경은 불가능하며, 군령을 초기화 한 후 재선택 해야한다.
